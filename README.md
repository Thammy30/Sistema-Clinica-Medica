# 🏥 Sistema Clínica Médica

Sistema web desenvolvido em Java (Servlets + JSP) com padrão MVC e DAO, command factory, relacionanmento de 1:1 e 1:N, etc... 
utilizando MySQL como banco de dados.
O sistema permite o gerenciamento de pacientes, médicos, consultas e prontuários em uma clínica médica, incluindo cadastro,
listagem, edição e exclusão de registros.
Além disso, implementa o padrão de projeto Decorator para o sistema de agendamento de consultas, onde envia as notificações para 
os pacientes com as informações da consulta.
